# Web-sites
Веб-сайты на различные темы и сложности (Websites on various topics and levels of difficulty)

### Aftefacts
> **Описание:** Сайт-лэндинг, один из первых проектов.
 - **Технологии:** `HTML` `CSS`
- **Особенности:** Базовая верстка, структура лендинга.

### SHOPGUIDE
> **Описание:** Тестовое задание, выполненное в рамках технического собеседования.
 - **Технологии:** `HTML` `CSS` `JavaScript`
- **Особенности:** 
  - ✅ Адаптивная вёрстка
  - ✅ Бегущая строка

### Web-Hosting (QLoud)
> **Описание:** Лэндинг компании предоставляющей услуги хостинга с функциональным калькулятором.
- **Технологии:** `HTML` `CSS` `JavaScript`
- **Особенности:** 
  - ✅ Адаптивная вёрстка
  - ✅ Калькулятор стоимости тарифа (логика на JS)

### ValCup
> **Описание:** Многостраничный сайт (с иммитацией бэкенда). Платформа для проведения турниров (от создания турнира до рабочего лобби матча).
- **Технологии:** `HTML` `CSS` `JavaScript`
- **Особенности:**
  - ✅ Многостраничная структура
  - ✅ Формы регистрации и авторизации
  - ✅ Личный кабинет и создание турниров
  - ✅ Модальные окна
